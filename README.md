# YouTube-Recommendation-Comment-Picker
YouTubeの視聴者コメントのうち、再生時間を含むものを拾います。他の視聴者がコメントでお勧めするシーンを知りたい時などに使えます
## 使用イメージ
拡張機能をクリックすると出現するポップアップに、現在見ている動画に対する他の視聴者のコメントが、再生時間を含むもののみ（一部抜粋）に絞って表示されます。
